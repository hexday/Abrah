📡 اسکنر CDN Edge IP — Cloudflare

این ابزار IP‌های عمومی Cloudflare (منتشرشده در cloudflare.com/ips) را تست می‌کند. IP‌هایی با پینگ پایین‌تر می‌توانند به عنوان CDN IP در کانفیگ‌های VMess/VLESS/Trojan با SNI استفاده شوند. مناسب برای یافتن IP تمیز جهت تونل‌زنی از طریق CDN.

# IP آدرس پینگ میانگین پینگ کمینه وضعیت امتیاز عملیات

💡 CDN IP چیست؟

IP های لبه CDN (Edge IP) آدرس‌هایی هستند که Cloudflare برای سرویس‌دهی عمومی استفاده می‌کند. با استفاده از این IP ها در کانفیگ VMess/VLESS همراه با SNI مناسب، ترافیک از طریق CDN عبور می‌کند.

⚙️ نحوه استفاده

IP با پینگ خوب را کپی کنید. در کانفیگ VMess/VLESS، فیلد «address» را با این IP جایگزین کنید. SNI و Host header را روی دامنه اصلی سرور تنظیم کنید.
